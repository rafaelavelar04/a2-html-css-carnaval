# Programação Web — Centro Universitário Barão de Mauá (baraoweb2026)

Este repositório reúne materiais, atividades e exemplos da disciplina de Programação Web do Centro Universitário Barão de Mauá (turma 2026). O objetivo é centralizar conteúdos de apoio, exercícios práticos e entregas para que os alunos possam estudar, praticar e submeter as tarefas da disciplina.

## Estrutura geral

- `atividades_1bi/` — atividades do 1º bimestre (cada atividade em sua própria pasta, ex.: `a1_git/`, `a2_intro_html_css/`).
- `a1_git/`, `a2_intro_html_css/` — pastas com enunciados (`*.md`), exemplos e, quando necessário, materiais complementares.

Cada pasta de atividade normalmente contém um arquivo `*_tarefa.md` com conteúdo complementar e a Atividade Prática a ser realizada.

## Como usar este repositório

1. Navegue até a pasta da atividade que deseja resolver e leia o arquivo do enunciado (ex.: `a2_tarefa.md`).
2. Use o código de exemplo como ponto de partida e/ou crie seus arquivos dentro da pasta da atividade.
3. Quando concluir a atividade, siga as orientações do enunciado para entrega (push para repositório da atividade no geral ou outro procedimento informado pelo professor).

Observação: muitas atividades podem ser executadas diretamente no navegador (abra o arquivo HTML) ou utilizando extensões/servidores locais simples no editor (ex.: Live Server no VS Code).

## Ferramentas recomendadas

- Editor: Visual Studio Code (ou outro de preferência)
- Controle de versão: Git e GitHub
- Navegador moderno para testes (Chrome, Firefox)

## Boas práticas

- Leia cuidadosamente o enunciado antes de começar.
- Faça commits frequentes com mensagens claras.
- Separe código de produção e exemplos didáticos em pastas distintas quando necessário.
- Documente decisões importantes no `README.md` da atividade ou em comentários.
